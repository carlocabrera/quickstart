# New Project Quick Start

**Steps**
---------

**Download or Pull This Repo**
	Top of this page you can see where it says clone or download

 **Install Node**
	https://nodejs.org/en/

**Download Editor**
	https://code.visualstudio.com/

 **Install & Update Node Packages** 

    
    npm install
    
    npm update


**Start Server**

    npm run watch
